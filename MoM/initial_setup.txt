1) Traverse inside MoM folder in terminal

2) Create new virtual environment and activate it

3) Run "pip install -r requirements.txt"

4) Run "pip3 install torch==1.8.2+cu111 torchvision==0.9.2+cu111 torchaudio===0.8.2 -f https://download.pytorch.org/whl/lts/1.8/torch_lts.html" if pytorch installation raises error with requirements.txt installation

5) Run  "python manage.py runserver"
